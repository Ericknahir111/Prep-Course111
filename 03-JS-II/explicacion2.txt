&&.
Este operador logico se utilizara para comparar y verificar ambas expresiones de forma en la cual, si las dos son verdaderas,
devolvera true, si alguna de las exoresiones es false, devolvera false.

NOT!
este operador hace que se devuelva lo contrario a lo que se indique.

operador ""||""este nos indica que si una de las expresiones es true devolvera true y si el false devolver false y viceversa.
